# Security Policy

Jarvis PC Local is currently a pre-MVP/private-alpha readiness foundation. There is no public production support channel, production installer, or supported public release build yet.

## Safe Reporting

Do not open issues or discussions that include secrets, tokens, logs containing secrets, `.env` contents, private keys, local databases, screenshots with tokens, credential material, or private user data.

Use redacted details only. A useful security report includes:

- Scope of the concern
- Affected component
- Steps to reproduce
- Expected behavior
- Actual behavior
- Redacted logs or screenshots only

## Current Security Boundaries

The current repository does not implement paid AI APIs, external account connectors, browser automation, email/social/payment actions, or a production installer.

Future connector manifests are placeholders unless separately planned and reviewed. Protected files such as secrets, keys, local databases, and runtime logs should not be committed or shared in reports.
