# Dashboard Section Search

Dashboard section search is a client-side usability layer for the local Jarvis dashboard.

## What It Adds

- A search input in Dashboard Home.
- Expand all, collapse all, and clear filter controls.
- Searchable metadata on major dashboard sections.
- View-only section toggles for collapsing or expanding sections.

## Boundaries

- No backend endpoint is added.
- No existing dashboard section is removed.
- Safety notes remain in each section and are visible whenever the section is expanded.
- No new action capability is added.
- Existing dashboard/LAN access behavior is unchanged.
