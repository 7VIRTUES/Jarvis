# Recent Activity / Audit Trail Runbook

Use the Recent Activity / Audit Trail Center when you need a quick local view of recent Jarvis metadata.

## Review Steps

1. Open the local dashboard.
2. Find `Recent Activity / Audit Trail`.
3. Use `Refresh activity timeline` to reload the bounded safe metadata view.
4. Review counts by type/status and recent item summaries.
5. Open safe metadata detail links only when you need one item's bounded metadata.

## Interpreting Results

The center is a visibility surface. It does not make approval decisions, retry work, mutate reports, or certify readiness.

If an item summary is redacted, review the original local workflow context through the appropriate owner-approved path rather than pasting secrets or raw logs into the dashboard.
