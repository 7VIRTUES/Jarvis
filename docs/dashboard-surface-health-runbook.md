# Dashboard Surface Health Runbook

Use the Dashboard Surface Health Center to inspect local dashboard wiring after adding or changing dashboard sections.

## Review Steps

1. Open the local dashboard.
2. Find `Dashboard Surface Health Center`.
3. Use `Refresh surface health` to reload the read-only health summary.
4. Review warning counts and per-surface warnings.
5. Use the API detail link only for safe metadata about one checked surface.

## Expected Warnings

A warning means a required section, guarded endpoint, docs link, summary key, or safety note was not found in the current local dashboard surface.

The center does not fix or certify anything. It is a visibility surface only.
