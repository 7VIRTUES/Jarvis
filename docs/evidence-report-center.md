# Evidence Report Center

The Evidence Report Center is a local-only report visibility surface for Jarvis PC Local. It indexes safe metadata from local Markdown and JSON reports in the Jarvis reports directory so a user can find evidence generated by existing agents without browsing arbitrary folders.

## What It Does

- Lists report metadata from the configured Jarvis reports directory.
- Infers report types from generated filenames.
- Extracts bounded Markdown headings and safe status fields.
- Extracts safe top-level JSON metadata.
- Redacts secret-like values, private-path shapes, and token-like strings before returning summaries or generated Markdown detail.
- Provides guarded dashboard/API access through the existing dashboard LAN access boundary.

## What It Does Not Do

- Does not upload, send, share, email, publish, delete, edit, rewrite, or certify reports.
- Does not scan arbitrary folders.
- Does not read protected secret files, raw databases, raw findings, raw notes, raw evidence, or raw logs.
- Does not run commands, call external services, write Git state, or perform browser, VirtualBox, installer, Tauri, pairing, OAuth, cloud sync, or telemetry work.

## Report Types

- `security-safety-*.md`: Security/Safety Review reports.
- `validation-*.md` or `validation-run-*.md`: Validation reports.
- `private-alpha-readiness-*.md`: Private-alpha readiness snapshots.
- `diagnostics-bundle-*.md` or `diagnostics-bundle-*.json`: Redacted diagnostics bundles.
- Other `.md` files: generic Markdown reports.
- Other `.json` files: generic JSON reports.

Unknown report types are indexed with warnings instead of treated as authoritative.

## Guarded Endpoints

- `GET /evidence/reports`: report index metadata only.
- `GET /evidence/reports/{report_id}`: generated safe Markdown detail or JSON metadata-only response.
- `GET /evidence/reports/{report_id}/metadata`: safe metadata for one report.

`report_id` must be a single filename. Path traversal, absolute paths, non-report extensions, hidden files, backups, oversized files, symlink escapes, and protected paths are blocked or skipped.

## Dashboard Surface

The dashboard shows report counts by type, recent report metadata, safe detail links, metadata links, endpoint references, and a safety note. The only control is refreshing the index.

## Safety Boundaries

The Evidence Report Center is local evidence visibility only. It is not production readiness, security certification, report publishing, report cleanup, credential rotation, history review, or raw evidence review.
