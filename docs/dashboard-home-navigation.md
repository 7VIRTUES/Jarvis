# Dashboard Home Navigation

The dashboard home section is a read-only navigation layer for existing Jarvis dashboard surfaces.

## What It Adds

- A top `Dashboard Home` section.
- Quick links to existing dashboard anchors.
- Compact status chips rendered from the existing dashboard summary payload.

## Boundaries

- No new backend endpoint is required.
- No existing safety notes are hidden or removed.
- No new action capability is added.
- Navigation labels use view-only language.
- Existing guarded dashboard/LAN behavior is unchanged.
